# ☕ Java POO - Base de Conhecimento

Este repositório é uma **base completa de estudos e experimentos em Java** desenvolvido na faculdade, com foco principal em **Programação Orientada a Objetos (POO)**.

Aqui têm:

* 📖 Anotações teóricas
* 💻 Exemplos práticos em código
* 🧠 Implementações de conceitos fundamentais de Java
* 🧩 Exercícios e experimentos

---

# 🎯 Objetivo

  Esse repositório foi criado para:
* Consolidar conhecimentos de **Java**
* Praticar **Programação Orientada a Objetos**
* Servir como **base de consulta pessoal**
* Armazenar **anotações importantes**
* Demonstrar conhecimento em **POO no GitHub**

---

# 📚 Conteúdo do Repositório

O repositório está dividido em duas partes principais:

### 📄 Anotações

Um arquivo `.txt` contendo:

* conceitos teóricos
* explicações de POO
* observações sobre Java
* dicas importantes
* resumos de aula

### 💻 Códigos Java

Uma pasta contendo vários arquivos `.java` com implementações práticas dos conceitos estudados.

Cada arquivo demonstra **um conceito específico da linguagem**.
